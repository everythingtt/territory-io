ToryScript Documentation

                        
Welcome, Architect. ToryScript is the high-level operational language of Territory.io. It allows you to sculpt worlds, manipulate economies, and orchestrate global warfare with surgical precision.

                    


                    
                    

                        

                            01
                            Core Syntax
                        

                        

                            
ToryScript uses a transpiled JavaScript syntax. The primary keyword is tory, which serves as a universal declarator for functions and variables.

                            
tory myBalance = 5000; // Variable declaration
tory setupMap() {      // Function definition
    reset();
    print("Initializing...");
}

                        

                    


                    
                    

                        

                            02
                            Graphics & Map API
                        

                        

                            

                                paint(x, y, w, h, type)
                                
Fills a rectangle. Types: 'land', 'water'. (Alias: rect)

                            

                            

                                line(x1, y1, x2, y2, type)
                                
Draws a precise line between two sets of coordinates.

                            

                            

                                river(x1, y1, x2, y2, width)
                                
Carves an organic water path with variable thickness.

                            

                            

                                cluster(x, y, size, type)
                                
Generates a natural-looking cluster of land or mountains.

                            

                            

                                seed(scale, threshold)
                                
Procedural generation using noise. Higher scale = smaller islands.

                            

                            

                                mountain(x, y, w, h)
                                
Places impassable terrain (ID: -3). Essential for chokepoints.

                            

                            

                                width(val) / height(val)
                                
Resizes the map grid and redraws all tiles.

                            

                            

                                tilesize(val)
                                
Changes the rendering size of each tile in pixels.

                            

                        

                    


                    
                    

                        

                            03
                            Game Logic & Hooks
                        

                        

                            

                                

                                    
config(key, [value])

                                    
Modify engine constants like 'CYCLE', 'CITY_BONUS', or 'STARTING_SIZE'.

                                

                                

                                    
tick(callback)

                                    
Hook into the 1-second game pulse.

                                

                                

                                    
cycle(callback)

                                    
Hook into the 10-second economy pulse.

                                

                                

                                    
claim(coords, owner)

                                    
Batch claim territory via [{x,y}, ...].

                                

                                

                                    
wait(ms)

                                    
Pause script execution for specified milliseconds.

                                

                                

                                    
stop()

                                    
Immediately terminate the current script session.

                                

                                

                                    
openFile()

                                    
Standalone Only: Open a native file picker to read map data.

                                

                                

                                    
saveFile(data, name)

                                    
Standalone Only: Save map data directly to the local filesystem.

                                

                                

                                    
maxEntities(n)

                                    
Sets the total number of entities (bots/players) for the next test.

                                

                                

                                    
gameMode(mode)

                                    
Sets mode: 'ffa' or 'team'.

                                

                                

                                    
spawnType(type)

                                    
Sets spawn: 'random' or 'selectable'.

                                

                                

                                    
difficulty(level)

                                    
Sets AI difficulty: 'ez', 'strategic', 'hard', or 'insane'.

                                

                                

                                    
playerName(name)

                                    
Sets the commander name for the player.

                                

                                

                                    
botNames([names])

                                    
Defines a custom pool of names for bots.

                                

                                

                                    
mod(name, value)

                                    
Modifies internal engine settings at runtime (e.g., 'cycle_time', 'ai_aggression').

                                

                            

                        

                    


                    
                    

                        

                            04
                            Entities & Diplomacy
                        

                        

                            

                                id(string)
                                
Finds an entity by its ID (e.g., 'pr12'). Returns an object with index and type.

                            

                            

                                bot(idx, [prop], [val])
                                
Deep access to bot state. Modify 'balance', 'difficulty', or 'color'.

                            

                            

                                war(host, target)
                                
Triggers a formal war declaration between two entity indices.

                            

                        

                    


                    
                    

                        

                            05
                            UI/UX Construction
                        

                        

                            

                                
Build custom interfaces directly inside the game container. ToryScript provides two distinct rendering layers for your UI elements:

                                

                                    

                                        
Map Layer ('map')

                                        
Elements scale and pan with the map. Best for territory markers or terrain labels.

                                    

                                    

                                        
HUD Layer ('hud')

                                        
Fixed-position elements that don't zoom. Ideal for menus, buttons, and custom status bars.

                                    

                                

                            


                            

                                

                                    html(str, [layer])
                                    Inject DOM ('map'|'hud')
                                

                                

                                    css(selector, style)
                                    Apply Styling
                                

                                

                                    removeUI(selector)
                                    Delete Element
                                

                                

                                    clearUI([all])
                                    Wipe UI Layer
                                

                                

                                    event(selector, type, fn)
                                    Interaction
                                

                                

                                    image(url, style, [layer])
                                    Static Media
                                

                                

                                    gif(url, style, [layer])
                                    Animated GIF
                                

                                

                                    audio(url, opts)
                                    MP3/Sound FX
                                

                                

                                    injectCSS(css)
                                    Global Styles
                                

                                

                                    injectJS(code)
                                    Raw Scripting
                                

                                

                                    update()
                                    Live UI Refresh
                                

                                

                                    overwrite(name, fn)
                                    Override Globals
                                

                                

                                    when(pred, fn)
                                    Conditional Run
                                

                                

                                    gamestate
                                    State Access
                                

                                

                                    renderTile(x, y)
                                    Single Tile Draw
                                

                                

                                    renderAll()
                                    Full Redraw
                                

                            


                            

                                

                                    🛡️
                                    

                                        
Safety & Garbage Collection

                                        
To prevent memory leaks and crashes, all injected UI elements are automatically purged when their parent script is stopped. Only elements created via ToryScript, Sandbox Engine components, or core game buttons (starting with btn-) can be styled or modified using css() or event().

                                    

                                

                            

                        

                    


                    
                    

                        

                            06
                            Integration & Console Guide
                        

                        

                            

                                

                                    
Chrome Developer Console

                                    
Directly interface with the engine. Every game mode supports hidden scripting capabilities via the ToryScript object.

                                    

                                        
ToryScript.help(); // View Full API

                                        
ToryScript.exec("toggleExplorer();");

                                    

                                

                                

                                    
Live Overwrite & Injection

                                    
Overwrite core game functions or inject global styles and scripts for deep engine modification.

                                    

                                        
overwrite('gameLoop', tory() { ... });

                                        
injectCSS(".tile.owned { border: 1px solid white; }");

                                    

                                

                            

                            

                                

                                    ⚠️
                                    

                                        
Architect Warning

                                        
Overwriting core functions like gameLoop or drawTile can cause game instability. Always backup your workspace before deep injection.

Real-time ownership map (IDs 0 to N).

Current development stages (hj0 to hj8).

                                

                                

                                    
tileFlags.live

                                    
Binary bitmasks for borders and dams.

Master the art of expansion, economy, and global dominance in this deep-strategy grid simulation.

                    


                    
                    

                        

                            
⚙️

                            
Core Mechanics

                        

                        

                            

                                
The Grid System

                                
The world is a high-resolution 384x216 grid. Every pixel is a tile that can be claimed, developed, or destroyed. The game engine uses optimized binary buffers to handle over 80,000 tiles in real-time.

                            

                            

                                
Economy & Income

                                
Your wealth is generated every second (Tick). Income scales with your territory size, city bonuses, and research levels. A large Cycle (10s) triggers major economic events and bot logic updates.

                            

                            

                                
Expansion Cost

                                
Claiming tiles isn't free. Each successful claim increases your base tile cost by $0.25. Crossing water (Bridges) costs 5x more than land. Strategy lies in efficient growth.

                            

                            

                                
Diplomacy & War

                                
Declare wars to seize land. Wars require a Justification period (cycles). During war, expansion is cheaper but you suffer a balance penalty. Alliances allow for shared vision and protection.

                            

                            

                                
Dams & Enclosures

                                
Enclose water bodies completely within your borders to create Dams. Dams provide a massive +$10 income bonus per cluster. It's the most efficient way to generate wealth without increasing tile costs.

                            

                            

                                
Capital Paralysis

                                
Losing your capital is catastrophic. Your empire will be Paralyzed for 15 cycles, meaning zero income and zero expansion capability. Rebuild it quickly via the 'Rebuild' menu.

                            

                        

                    


                    
                    

                        

                            
🧬

                            
Research & Tech

                        

                        

                            
Modernize your empire by spending Research Tokens. Tokens are earned slowly over time or through specific investments.

                            

                                

                                    
Economic Tree

                                    
Reduce tile costs, increase max balance, and lower war penalties.

                                

                                

                                    
Military Tree

                                    
Unlock Missiles and Warplanes. Increase Air Support range and annex radius.

                                

                                

                                    
Political Tree

                                    
Manage alliances and internal stability. (Coming soon in v3.3)

                                

                            

                        

                    


                    
                    

                        

                            
📊

                            
Management & Investments

                        

                        

                            

                                
City Bulk Management

                                
Access the Management Modal to view all your cities at once. Monitor their population, levels (hj), and development progress. You can bulk-develop cities if you have the balance.

                            

                            

                                
Strategic Investments

                                
Invest your surplus balance into short-term projects. These provide massive income multipliers or token generation for a fixed duration. Choose between 'Scale' and 'Static' returns.

                            

                        

                    


                    
                    

                        

                            
⚔️

                            
Warfare & Assault Planning

                        

                        

                            

                                
Assault Planning Mode

                                
Don't just attack—plan. Use the Right-Click context menu on an enemy to enter Planning Mode. Select multiple target tiles to queue up a massive simultaneous assault.

                            

                            

                                

                                    
Air Support captures

                                    
Use Warplanes to capture tiles deep behind enemy lines. Requires Military Research Level 2+.

                                

                                

                                    
Capital Destruction

                                    
Targeting an enemy capital during war will paralyze their expansion for several cycles.
