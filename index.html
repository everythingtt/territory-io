ToryScript API

ToryScript is a JavaScript-based API for generating custom maps in Territory.io.

fill(x, y, width, height, type)

Fills a rectangular area with a specific tile type.

Parameters:

  • type: 'water' (0) or 'land' (1)
circle(x, y, radius, type)

Creates a circular shape of tiles.

spawnCity(x, y)

Places a city at the specified coordinates.

spawnCapital(x, y)

Sets the player starting capital location. Only one capital can exist.

noise(scale, threshold)

Generates procedural terrain across the entire map using sine-based noise.

clear()

Resets the entire map to Land tiles.

log(message)

Outputs a message to the engine console.

Advanced Example: Archipelago

// Reset map
clear();

// Generate base terrain
noise(0.15, 0.4);

// Add custom islands
circle(100, 100, 30, 'land');
circle(250, 150, 20, 'land');

// Place cities
spawnCity(100, 100);
spawnCity(250, 150);

// Set start point
spawnCapital(100, 100);

log("World Generated!");
